Mobile Connect

A mobile-based digital identity service enabling secure authentication and authorization for users and non-human identities.

Description

Mobile Connect is a standardized mobile authentication service that allows users to access online services securely through their mobile devices. It leverages the existing mobile network infrastructure to provide a seamless experience for both human users and non-human identities (NHIs) such as IoT devices and applications. By using Mobile Connect, NHIs can authenticate themselves using the credentials linked to a mobile number, simplifying the process of identity verification and access control. This is particularly important in the context of IoT, where many devices need unique identities to connect and communicate securely with other systems. Mobile Connect also enhances user privacy by allowing users to control what information is shared with service providers, while also ensuring that NHIs can operate within a secure framework. The service is being adopted globally, facilitating secure interactions across various sectors including finance, healthcare, and smart cities.

Examples

  • An IoT sensor using Mobile Connect to authenticate itself to a cloud service.
  • A smart home device that allows users to manage settings through a secure mobile app using Mobile Connect.

Additional Information

  • Mobile Connect is supported by various mobile network operators worldwide.
  • It enhances trust in digital transactions by providing a secure and user-friendly authentication method.
